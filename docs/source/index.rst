SlyPI
=====

SlyPI is a Python interface for Slycat.  Slycat is a web application for interactive 
visualization of ensemble data (see https://github.com/sandialabs/slycat and 
https://slycat.readthedocs.io/en/latest).  SlyPI contains code that enables 
you to interact with the Slycat web server through a Python interface.  You can 
query the Slycat server, create Slycat models, and use the code to integrate your 
own algorithms into the Slycat pipeline.
